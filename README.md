RD NEO
Software sviluppato da fmarsico70.

Installazione
  Puoi installare RD NEO in due modi diversi:
    1. Tramite Windows Package Manager (Consigliato)
    Questo è il metodo più sicuro. Microsoft scansiona il pacchetto prima di renderlo disponibile. Apri il Terminale (PowerShell) e digita:
      winget install fmarsico70.rd_neo
      (Nota: Se il comando non viene riconosciuto, aggiorna l'App Installer dal Microsoft Store).
    2. Download Diretto (.exe)
      Se preferisci scaricare l'eseguibile manualmente:
      Vai nella sezione Releases di questo repository.
      Scarica il file rd_n.exe.
  Al primo avvio, se Windows mostra l'avviso SmartScreen:
    Clicca su "Ulteriori informazioni".
    Clicca su "Esegui comunque".
